Cuestionario Laboratorio 1

1. ¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet: Es una red global de computadores interactuando que se comunican entre sí mediante protocolos de comunicación estñandar. Adicional a esto, esta es una infraestructura físicaque permite transferir datos entre dispositivos.

World Wide Web (WWW): Es un sistema de información en linea, esta opera sobre internet y permite el acceso a documentos mediante enlaces de hipertexto. Este es uno de los muchos servicios de internet disponibles. Hace uso de HTPP para la transferencia de datos, HTML para la presentación del contenido.

2. ¿Cuáles son las partes de un URL?

Protocolo, Subdominio, Dominio, Dominio de nivel superior, Subcarpeta, Slug

3. ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET: Solicita datos de un recurso especificado.

HEAD: Similar al GET, pero este responde solo a los encabezados de la respuesta y sin cuerpo del mensaje.

POST: Envía datos al servidor para ser procesados.

PUT: Envía datos al servidor para actualizar o crear recursos en una ubicación específica.

PATCH: Similar al PUT, pero este se utiliza para modificar parcialmente en un recurso.

DELETE: Elimina un recurso especificado en el servidor.

4. ¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

POST, es el más adecuado para enviar formularios HTML que pueden contener datos sensibles, esto debido a sus caracteristicas de seguridad, capacidad de envio de datos y de procesamiento del servidor.

7. ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

Sí, es responsabilidad del desarrollador corregir un sitio web si un usuario reporta un error 404 (Página no encontrada) al intentar acceder al sitio. Sin embargo tambien dependera del nivel de seguridad que se quiera tener del sitio para evitar hackeos.

significa que aunque aún es compatible con los navegadores y funciona correctamente, se recomienda no utilizarlo en nuevos desarrollos web.

10. ¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

Nuevos elementos semánticos que permiten mejor estructuración y descripción del contenido. Compatibilidad con multimedia, en HTML5 incluye etiquetas especificas para la reproducción de contenido multimedia. Gráficos y animación, en HTML5 existe el elemento canvas que permite dibujar gráficos, animaciones y otros elementos mediante JavaScript.

11. ¿Qué componentes de estructura y estilo tiene una tabla?

Estos son algunos componentes de estructura de una tabla. Table, este es el elemento principal para definir la tabla en HTML. Thead, define la sección del encabezado de la tabla. Tbody, define la sección del cuerpo de la tabla. Tfoot, define la sección del pie de página de la tabla. Tr, define una fila en la tabla. Th, define una celda de encabezado en la tabla. Td, define una celda de datos en la tabla.

Para el estilo de esta tabla se hace uso de atributos de estilo CSS, Clases y IDs.
